Приступая к работам по теме Java Server Pages, многие из вас совершают одну очень серьёзную ошибку, а именно:

Вы располагаете JSP страницы в клиентской части приложения

В некоторых задачах есть специальное требование, по которому вы должны положить JSP-страницы в клиентскую часть. Это сделано для того, чтобы вы просто потренировались и увидели — да, так можно делать. Но, убедившись, что Так можно сделать, мы потом много времени тратим на то, чтобы доказать, что:

ТАК ДЕЛАТЬ НЕЛЬЗЯ!

Поэтому, если расположение JSP в клиентской части приложения не требуется условиями задачи, то мы используем одно простое правило:

JSP страницы должны располагаться в серверной части приложения!
