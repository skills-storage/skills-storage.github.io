В ситуации, когда вам нужно сравнить две строки на равенство символов, вы используете оператор сравнения ==.

Это ошибка, так как вы забываете, что строка в Java является объектом типа java.lang.String.

Поэтому, как и для любого другого объекта, для сравнения строк вы должны использовать метод java.lang.Object.equals(Object)
