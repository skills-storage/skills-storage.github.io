Напиши о том, что хочется использовать ==, потому что строку можно объявить, как и любой другой примитивный тип. Но это только лишь - синтаксический сахар! На самом деле строка - это объект! Более того, если вы инициализируете сроку через ==, то результат сравнения будет один. а если через объект - то другой! И в этом проблема. И ссылки на упражнения дай.
