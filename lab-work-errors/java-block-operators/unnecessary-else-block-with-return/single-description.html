В принципе, для краткого описания этой ошибки, достаточно ее названия. Иногда нашей программе нужно выбрать одно значение из двух вариантов - для этого Вы используте конструкцию if-else. Но если в блоке if Вы понимаете что дальнейшее выполнение текущего метода Вам больше не требуется, и решате вернуть значение из блока if, То тогда возникает вопрос - Зачем Вам блок Else?
