Здесь имеются ввиду символы табуляции, которые вы используете для того, чтобы сдвинуть внутренний блок кода относительно внешнего. Все мы знаем простое правило:

Вложенный блок кода должен смещаться относительно своего внешнего блока на 4 пробела или один TAB

Так вот пришло время добавить еще одно правило:

Запрещено использовать TAB для смещения вложенных блоков кода!

И тогда финальное правило будет звучать так:

Вложенный блок кода должен смещаться относительно своего внешнего блока на 4 пробела

Ну а с чем это связано, вы cможете прочитать в более подробном описании ошибки. Если, конечно, оно вам надо и у вас есть время. С другой стороны, там будет написано, как настроить разные редакторы для того, чтобы они автоматически заменяли TAB-ы пробелами. Так что читать, скорее всего, придется.
