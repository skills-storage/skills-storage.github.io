Кто вы — человек или продукт (бренд)?

Второй самый главный вопрос: Каким должно быть имя пользователя?

Дальше ебейший пъяный пиздец (мысли вслух)

Потому что имя пользователя будет отображаться в вашем URL-е Потому что имя пользователя будет отображаться на главной странице вашего аккаунта (проверь!) Потому что ваше имя пользователя будет использоваться в пул реквестах (проверь) Потому что ваше имя пользователя будут использовать, чтобы добавить вас коллаборатором в свой проект Короче - имя пользователя - это ваше имя на гитхаб. То есть то, кем вас будут видеть другие люди. Естественно, выбор правильного имени - это важнейший шаг в вашей жизни! К счастью, он не такой сложный, как кажется на первый взгляд :) Все что вам нужно - это определиться 1. Это ваш персональный GitHub аккаунт 2. Или это аккаунт вашего продукта?
