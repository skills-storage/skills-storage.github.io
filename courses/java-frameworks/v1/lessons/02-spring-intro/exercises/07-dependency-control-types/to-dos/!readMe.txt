Это должно быть первое упражнение по списку!
Или второе, если ты примешь решение все же использовать упражнение - "no-spring-app", которое будет идти слеующим.
В этом упражнении рассматриваются разные способы организации управления зависимостями, на основе классических JEE технологий.
Эти способы включают:
1. Прямой контроль - когда ты сам создаешь объекты.
2. Dependency Lookup - когда депенденси уже есть, но тебе их нужно "найти".
3. Dependency Injection - когда депенденси уже есть и тебе их просто дают.
