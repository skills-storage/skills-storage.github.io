Эта ошибка говорит нам о том, что необходимо добавить Servlet API Maven Dependency в наш проект. Для этого вы можете выполнить отдельное упражнение (ссылка), или мы прямо здесь приведем краткую инструкцию: 1. Перейдите на сайт mvnrepository.com 2. найдите сервлеты введя артефакт в строку поиска 3. выберите версию - мы будем использовать 3.0.1 4. Скопируйте XML-секцию для депенденси 5. вставьте ее в pom.xml Тогда наш пом.xml будет выглядеть вот так: картинка и ошибка компиляции не пропадет. Примечание. Иногда, вы сделали все правильно - в нашем случае добавили нужные депенденси, а синтаксические ошибки все равно не пропадают В этом случае сделайте maven -> update project...
