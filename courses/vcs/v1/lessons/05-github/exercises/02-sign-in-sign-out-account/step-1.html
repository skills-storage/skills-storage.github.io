Кнопок главного меню – всего три. Чтобы их увидеть, посмотрите в правый верхний угол сайта GitHub. Вот эти кнопки:

«Не по-програмисстски» их можно назвать:

  1. Колокольчик
  2. Плюсик
  3. Какой-то круг

Давайте подберем для них «програмистские» имена:

  1. Колокольчик это – GitHub Notification – уведомления от сайта GitHub
  2. Плюсик это – Add New – возможность быстро что-то добавить
  3. Какой-то круг – это основные функции вашего аккаунта

Давайте посмотрим на каждую кнопку в отдельности.
